# WhatsOrder

WhatsOrder is a modern web application designed for business management and order processing. Built with a focus on speed and scalability, it leverages the latest web technologies to provide a seamless experience for both business owners and their customers.

## üöÄ Features

- **User Authentication**: Secure signup, login, and password management systems.
- **Onboarding Flow**: Smooth guided setup for new businesses.
- **Business Management**: Tools to manage your business profile and presence.
- **Interactive Dashboard**: Real-time overview of business activities and metrics.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop environments.

## ‚öôÔ∏è Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/whatsorder-ng.git
    cd whatsorder-ng
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open your browser**:
    Navigate to `http://localhost:5173` (or the port specified by Vite).
